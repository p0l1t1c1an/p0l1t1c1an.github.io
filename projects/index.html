Mines

This is one of my first ever C programming projects that I worked on. It was originally a text based program but I made a GTK3 interface for a good ol' final project. There's definitely a multitude of issues with such a program and ultimately I would love to completely overhaul this sloppy program. But it works for when I just want to play minesweeper every once and a while. Ultimately, building this program taught me a lot about C development and low level memory management, which I now find incredibly fun to work with.

gitstatus-rs

This is a simple program that I use to see information about the current git repo that I am working in. It is meant to mimick the gitstatus integrated into powerlevel10k. Gitstatus-rs isn't quite a suitable replacement for the original, but should work on systems supporting Rust. This means that there is a working option for my OS of choice OpenBSD, which the original does not support.
